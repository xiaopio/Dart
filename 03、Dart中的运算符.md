在 Dart 中，运算符本质上是类中定义的方法。例如，当你使用 `a + b` 时，实际上是调用了 `a` 对象的 `+` 方法。

------

## 1. 算术运算符 (Arithmetic Operators)

用于执行标准数学运算。

| **运算符** | **说明**                 | **示例**             |
| ---------- | ------------------------ | -------------------- |
| `+`        | 加法                     | `a + b`              |
| `-`        | 减法                     | `a - b`              |
| `*`        | 乘法                     | `a * b`              |
| `/`        | 除法（结果总是浮点数）   | `10 / 3 // 3.333...` |
| `~/`       | **整除**（结果为整数商） | `10 ~/ 3 // 3`       |
| `%`        | 取模（余数）             | `10 % 3 // 1`        |

------

## 2. 关系运算符 (Relational Operators)

用于比较两个对象。

| **运算符** | **说明** | **示例** |
| ---------- | -------- | -------- |
| `==`       | 等于     | `a == b` |
| `!=`       | 不等于   | `a != b` |
| `>`        | 大于     | `a > b`  |
| `<`        | 小于     | `a < b`  |
| `>=`       | 大于等于 | `a >= b` |
| `<=`       | 小于等于 | `a <= b` |

------

## 3. 类型判定运算符 (Type Test Operators)

在运行时检查或转换对象的类型。

| **运算符** | **说明**                            | **示例**                      |
| ---------- | ----------------------------------- | ----------------------------- |
| `as`       | 类型转换 (Typecast)                 | `(p as Person).name = 'Bob';` |
| `is`       | 如果对象是指定类型则返回 true       | `if (p is Person) { ... }`    |
| `is!`      | 如果对象**不是**指定类型则返回 true | `if (p is! Person) { ... }`   |

------

## 4. 赋值运算符 (Assignment Operators)

除了基础的 `=`，Dart 还支持复合赋值和**空赋值**。

| **运算符**          | **说明**                             | **示例**                      |
| ------------------- | ------------------------------------ | ----------------------------- |
| `=`                 | 基础赋值                             | `a = value;`                  |
| `??=`               | **空赋值**：仅当变量为 null 时才赋值 | `b ??= value;`                |
| `+=`, `-=`, `*=`... | 复合赋值                             | `a += 5; // 等同于 a = a + 5` |

------

## 5. 逻辑运算符 (Logical Operators)

用于组合布尔表达式。

| **运算符** | **说明**     | **示例**                 |
| ---------- | ------------ | ------------------------ |
| `!expr`    | 逻辑非 (NOT) | `!true // false`         |
| `          |              | `                        |
| `&&`       | 逻辑与 (AND) | `true && false // false` |

------

## 6. 条件表达式 (Conditional Expressions)

Dart 提供简写方式来替代简单的 `if-else`。

- **三元运算符**：`condition ? expr1 : expr2`
  - 如果条件为真，执行 `expr1`，否则执行 `expr2`。
- **空合并运算符**：`expr1 ?? expr2`
  - 如果 `expr1` 不为 null，返回其值；否则返回 `expr2` 的值。

------

## 7. 级联操作符 (Cascade Notation)

级联操作符 (`..` 和 `?..`) 允许你在**同一个对象**上执行一系列操作。这在配置复杂对象时非常有用。

Dart

```dart
var paint = Paint()
  ..color = Colors.black
  ..strokeCap = StrokeCap.round
  ..strokeWidth = 5.0;
```

- `..`：普通级联。
- `?..`：空安全级联（如果对象为 null，则跳过后续操作）。

------

## 8. 其他运算符

| **运算符** | **说明**                        | **示例**       |
| ---------- | ------------------------------- | -------------- |
| `.`        | 成员访问                        | `person.name`  |
| `?.`       | 条件成员访问（空安全访问）      | `person?.name` |
| `[]`       | 访问列表或 Map 索引             | `list[0]`      |
| `!`        | 非空断言（强制认定变量非 null） | `text!.length` |

------

## 运算符优先级 (Precedence)

在复杂的表达式中，运算符的执行顺序如下（从高到低）：

1. 一元后缀 (`.`, `?.`, `()`, `[]`)
2. 一元前缀 (`-`, `!`, `~`, `++`, `--`)
3. 乘除类 (`*`, `/`, `~/`, `%`)
4. 加减类 (`+`, `-`)
5. 移位类 (`<<`, `>>`)
6. 关系类 (`<`, `>`, `<=`, `>=`)
7. 相等类 (`==`, `!=`)
8. 逻辑与 (`&&`)
9. 逻辑或 (`||`)
10. 三元/空合并 (`??`, `? :`)
11. 赋值 (`=`, `+=`, `??=` 等)

------

### **小贴士**

在 Dart 中，如果你想要判断两个变量是否引用同一个对象，应该使用 `identical(a, b)` 函数，而 `==` 通常用于检查“值”是否相等（取决于类是否重写了 `==` 运算符）。
